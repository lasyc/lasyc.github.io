HTML 简介

HTML 是 HyperText Markup Language 的缩写，中文翻译为超文本标记语言。通过标签来标记、描述网页内容。

HTML 标签的介绍

HTML 标签是 HTML 文档的构建块，用于定义网页的元素。

标签按照格式分为：单标签和双标签。

单标签：如<br />、<hr />、<img src="xxx" /> 等只有开始标签，没有结束标签。
双标签：如<p></p>、<div></div>、<h1></h1> 等，由开始标签和结束标签组成。

个别标签还拥有属性，如 <img src="xxx" /> 中 src 属性用于指定图片的源文件。

个别标签还拥有事件，如 <button onclick="xxx()"></button> 中 onclick 属性用于指定按钮点击时执行的 JavaScript 代码。

标签的语法

  1. 双标签内可以嵌套其他标签，但是不可以交叉嵌套：

    • 正确：<div><p>文字</p></div>
    • 错误：<div><p>文字</div></p>

  2. 双标签必须关闭：

    • 正确：<div>文字</div>
    • 错误：<div>文字

  3. 单标签也需要关闭：

    • 正确：<br />
    • 错误：<br>

  4. 标签中的属性必须使用引号括起来：

    • 正确：<img src="xxx" />
    • 错误：<img src=xxx />

  5. 注释不能嵌套：

    • 正确：<!-- 注释 -->
    • 错误：<!-- <!-- 注释 --> -->

字体标签 font（HTML5 已弃用）

font 标签在 HTML 4 中用于规定字体、字体大小和文本颜色。

在 HTML 5 中，font 标签已被弃用M，使用 p 标签代替，通过 style 属性中的 font-family 属性来设置字体样式。

在 HTML 中，< 和 > 等字符是具有特殊含义的字符，我们无法直接将其在浏览器中显示。此时就需要使用实体来替换。

字符 实体

‘ &apos;

全角空格 &emsp;

标题标签

标题标签是 h1 ~ h6，从大到小排列。

超链接标签

在网页中，通过点击超链接可以跳转到其他页面。超链接标签为 a。

属性 值 描述
download 文件名 规定当用户单击超链接时将下载目标。
href URL 规定链接指向的页面的 URL。
hreflang 语言代码 规定被链接文档的语言。
media 媒体查询 规定被链接文档是为何种媒介/设备优化的。
ping URL 列表 规定以空格分隔的 URL 列表，当链接被访问时，浏览器将发送带有 ping 正文的 POST 请求（在后台发送）。通常用于跟踪。
referrerpolicy 引用信息 规定要与链接一起发送的引用信息。
rel 关系 规定当前文档和被链接文档之间的关系。
target 目标 规定在何处打开链接。
type 类型 规定链接的 MIME 类型。

列表标签

列表标签分为有序列表和无序列表。有序列表为 ol，无序列表为 ul，其中的每一项都是 li 标签。

属性 值 描述
reversed reversed 规定列表顺序应该反转（9,8,7 …）。
start 数字 规定有序列表的起始值。
type 1、A、a、I、i 规定要在列表中使用的标记类型。

img 标签

img 标签用于在 HTML 页面中嵌入图像。

属性 值 描述
alt 文本 规定图像的替代文本。
crossorigin anonymous、use-credentials 允许使用来自允许跨域访问的第三方网站的图像与画布（canvas）一起使用。
height 像素 规定图像的高度。
ismap ismap 将图像定义为服务器端图像映射。
loading eager、lazy 指定浏览器是否应立即加载图像，或者推迟加载图像直到满足某些条件。
longdesc URL 规定指向图像详细描述的 URL。
referrerpolicy no-referrer、no-referrer-when-downgrade、originorigin-when-cross-origin、unsafe-url 规定在获取图像时要使用的引用信息。
sizes 尺寸 规定不同页面布局的图像尺寸。
src URL 规定图像的路径。
srcset URL-list 规定在不同情况下使用的图像文件列表。
usemap #mapname 将图像定义为客户器端图像映射。
width 像素 规定图像的宽度。

表格标签

table 标签定义了 HTML 表格。

一个 HTML 表格由一个 table 元素和一个或多个 tr、th 和 td 元素组成：

  • tr：元素 定义表格行
  • th：元素 定义表格标题
  • td：元素 定义表格单元格

HTML 表格还可以包含以下元素：

  • caption：定义表格标题
  • colgroup：定义表格列
  • thead：定义表格头部
  • tfoot：定义表格尾部
  • tbody：定义表格的主体

跨行跨列表格

在一些情况下，我们需要将多个单元格合并为一个单元格。这时，我们可以使用 colspan 和 rowspan 属性。

colspan 属性用于合并列，rowspan 属性用于合并行。

iframe 标签

iframe 标签定义行内框架（内联框架）。行内框架用于在当前 HTML 文档中嵌入另一个文档。

属性 值 描述
allow 规定 <iframe> 的功能策略。
allowfullscreen true、false 如果 <iframe> 可以通过调用 requestFullscreen() 方法激活全屏模式，则设置为 true。
allowpaymentrequest true、false 如果允许跨源 <iframe> 调用 Payment Request API，则设置为 true。
height 像素 规定 <iframe> 的高度。默认高度为 150 像素。
loading eager、lazy 规定浏览器是应立即加载 iframe 还是推迟加载 iframe，直到满足某些条件为止。
name 文本 规定 <iframe> 的名称。
referrerpolicy 略 规定在获取 iframe 时要发送的引用信息。
sandbox allow-forms、allow-pointer-lock、allow-popups、allow-same-origin、allow-scriptsallow-top-navigation 启用一系列对 <iframe> 中内容的额外限制。
src URL 规定要嵌入到 <iframe> 中的文档的地址。
srcdoc HTML 代码 规定要在 <iframe> 中显示的页面的 HTML 内容。
width 像素 规定 <iframe> 的宽度。默认宽度为 300 像素。

form 标签用于为接收用户输入创建 HTML 表单。

属性 值 描述

action URL 规定提交表单时将表单数据发送到哪里。

enctype application/x-www-form-urlencoded、multipart/form-data、text/plain 规定在向服务器提交表单数据时，应该如何对表单数据进行编码。
method get、post 规定发送表单数据时使用的 HTTP 方法。
name 文本 规定表单的名称。
novalidate novalidate 规定提交表单时不应验证表单。
rel 略 规定链接资源和当前文档之间的关系。
target _blank、_self、_parent、_top 规定在何处显示提交表单后收到的响应。

  • optgroup：选项组标签
  • fieldset：表单字段集标签
  • label：表单项目标题标签

表单格式化

使用 table 标签来格式化表单。

表单提交细节

通过 action 属性来指定表单提交的目标地址，通过 method 属性来设置表单提交方式。

method 请求分为两种：分别是 get 和 post。

当使用 get 方式提交表单时，参数会以name=value的格式拼接到 url 中，且参数长度有限制。

当使用 post 方式提交表单时，参数会以 json 的格式放在请求体中，且参数长度没有限制。

  1. 所有表单标签必须包含在 form 标签内，否则不会被提交。
  2. 所有表单标签必须包含 name 属性，否则不会被提交。
  3. 单选框、复选框、option 标签必须包含 value 属性。

其他标签

在 HTML 中，除了上述一些标签，还有很多很多标签，如：div、span、p、button 等，具体可查看在线文档。

div 和 span 标签是一种容器标签，用于页面布局。但是 div 是块级元素，在页面中独占一行，而 span 是行内元素，可以在一行中显示多个。

CSS 技术

CSS 技术介绍

HTML 是一个描述网页内容的语言，而 CSS 是一个描述网页样式的语言。

在线文档

CSS 语法规则

  1. 选择器：用于匹配元素。
  2. 属性：用于描述元素样式。
  3. 值：属性的取值。

CSS 与 HTML 的结合方式

  1. 通过 style 属性写在标签中，称为行内样式。
  2. 通过 style 标签写在 head 中，称为内部样式。
  3. 创建 .css 文件，通过 link 标签引入，称为外部样式。

结合方式一

结合方式二

使用行内样式的时候如果样式过多，全部写在标签中会显得代码很乱，并且如果多个表现样式相同，代码需要些很多次，所以我们可以使用内部样式。

使用内部样式需要定义选择器，通过选择器来指定哪些标签采用该样式。我们可以将 CSS 选择器分为五类：

  1. 简单选择器（根据名称、id、类来选取元素）
  2. 组合器选择器（根据它们之间的特定关系来选取元素）
  3. 伪类选择器（根据特定状态选取元素）
  4. 伪元素选择器（选取元素的一部分并设置其样式）
  5. 属性选择器（根据属性或属性值来选取元素）

结合方式三

对于内部样式，虽然解决了行内样式的缺点，但是也存在一个缺点，那就是所编写的 CSS 代码仅在当前 HTML 文件中生效，并且将 CSS 代码和 HTML 代码混合在一起，使得代码的维护性变得非常困难。因此可以使用外部样式。

外部样式就是将 CSS 代码保存在单独的文件中，并引入到 HTML 文件中。

标签名选择器

上边提到了类选择器，其实 CSS 中还有很多种选择器，比如标签选择器，ID选择器，属性选择器等等。

标签选择器，就是通过标签名来选择元素。比如，div、span、p等等。

当以标签为单位来设置样式时，可以使用标签选择器。

ID 选择器

标签选择器会使所有标签都生效，但是如果想区分开来，可以使用ID 选择器。

通过 # + id 来定义 ID 选择器，在标签中通过 id 属性来指定选择器名称。

但是 id 属性的值不可以重复，因此使用 ID 选择器只能将样式应用于一个标签，多个标签不可复用。

类选择器

类选择器相比于ID 选择器，可以重复使用，并且类选择器可以添加多个类名，多个类名之间用空格隔开。

组合选择器

组合选择器由多个选择器组合而成，分为：

选择器 示例 示例描述
后代选择器 (空格) div p 选择 <div> 元素内的所有 <p> 元素。
子选择器 (>) div > p 选择其父元素是 <div> 元素的所有 <p> 元素。
相邻兄弟选择器 (+) div + p 选择所有紧随 <div> 元素之后的 <p> 元素。
通用兄弟选择器 (~) p ~ ul 选择前面有 <p> 元素的每个 <ul> 元素。
