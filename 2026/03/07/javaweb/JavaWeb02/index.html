JavaScript 介绍

JavaScript 是一种脚本语言，是一种运行在浏览器上的脚本语言。

JavaScript 和 HTML 代码的结合

方式一

使用 <script> 标签，来讲 js 代码编写到 html 文件中

方式二

将 js 代码编写到 js 文件中，并引入 html 文件中

变量

js 是一种弱类型的语言，变量声明时不需要指定变量的类型。

基本数据类型：

数据类型 描述
Number 数值
String 字符串
Boolean 布尔值
Null 空值
Undefined 未定义
Symbol 符号

引用数据类型：

数据类型 描述
Object 对象
Array 数组
Function 函数
Date 日期
RegExp 正则表达式

因为 js 定义变量时不需要声明数据类型，所以当我们想要查看变量的数据类型时，可以使用 typeof 运算符。

在 ES5 的时候可以使用 var 声明变量，但是 ES6 引入了 let 和 const 两种声明变量的方式。

  • let 声明的变量只在当前作用域有效，在当前作用域外无法访问。
  • const 声明的变量必须初始化，且不能被重新赋值。

关系运算符

同 Java 一样，关系运算符也分为等于、大于、小于、大于等于、小于等于、不等于。

运算符 示例 描述
== a == b 等于
> a > b 大于
< a < b 小于
>= a >= b 大于等于
<= a <= b 小于等于
!= a != b 不等于

但是 js 中，比 Java 多了一个 === 运算符，用于判断两个值是否相等，并且类型也相等。

运算符 示例 描述
=== a === b 严格等于
!== a !== b 严格不等

逻辑运算符

也同 Java 语言一样，逻辑运算符也分为与、或、非三种。

运算符 示例 描述
&& true && true true
| | true || false true
! !true false

并且在 js 中，所有的变量也可以作为一个 boolean 类型的值来使用，除了 0、null、undefined 空字符串以外所有的值均为 true

js 中的逻辑运算符的返回值也和 Java 中的有所区别

  1. &&：逻辑与，返回第一个为 false 的值，否则返回最后一个值。
  2. ||：逻辑或，返回第一个为 true 的值，否则返回最后一个值。

函数重载

在 js 中，不存在函数重载的概念。因为在 js 中，函数也是一种变量的值，所以如果定义多个同名函数，就相当于给变量赋值了多个值，最终只会保留最后一个值。

JS 中的事件

事件就是用户行为，比如点击、鼠标移动、键盘按下等。

事件分为：

  1. 鼠标事件
  2. 键盘事件
  3. 框架/对象事件
  4. 表单事件
  5. 剪切板事件
  6. 打印事件
  7. 拖动事件
  8. 多媒体事件
  9. 动画事件
  10. 过渡事件
  11. 其他事件

想要通过对应的行为触发对应的事件，首先我们要先注册事件，分为静态注册和动态注册。

  • 静态注册：在编写 HTML 标签时，通过 onXXX 属性来注册事件，如：onclick="clickHandler()"
  • 动态注册：需要先获取标签对象，然后调用 addEventListener() 方法来注册事件，如：button.addEventListener("click", clickHandler);

onload

onload 是一个事件，当页面加载完成后，会触发该事件。

onclick

onclick 是一个事件，当用户点击某个元素时，会触发该事件。

onfocus 和 onblur

onfocus 和 onblur 是 HTML 中的事件属性，用于处理元素获得焦点和失去焦点时触发的事件。

onchange

onchange 事件在元素失去焦点并发生改变时触发。

onsubmit

onsubmit 属性规定当提交表单时运行的脚本。

DOM 模型

DOM（Document Object Model）是 HTML 和 XML 文档的编程接口。DOM 是 W3C（World Wide Web Consortium）组织定义的。DOM 模型将 HTML 文档或 XML 文档作为树结构。树中的每个节点（node）都表示文档中的元素（element）。

Document 对象

Document 对象表示 HTML 文档。Document 对象是 DOM 树中的根节点。

通过 Document 对象，我们可以访问 HTML 文档中的所有元素。例如，我们可以通过 Document 对象获取 HTML 文档的标题、链接、表格、图像等等。

其中的每一个 DOM 元素都是一个对象。

常用方法

document 对象提供了以下常用的方法：

方法 描述
getElementById(id) 返回带有指定 id 的第一个元素
getElementsByTagName(tagName) 返回带有指定标签名的所有元素
getElementByClassName(className) 返回带有指定 class 的所有元素
getElementsByName(name) 返回带有指定 name 的所有元素
getElementsByTagNameNS(namespaceURI, localName) 获取指定命名空间下的指定标签名的所有元素
createElement(tagName) 创建一个元素节点
querySelector(selector) 返回匹配指定 CSS 选择器的第一个元素
querySelectorAll(selector) 返回匹配指定 CSS 选择器的所有元素

getElementById

getElementById 方法用于获取指定ID的元素，因为 id 属性是唯一的，所以 getElementById 方法返回的元素只有一个。

getElementsByName

getElementsByName 方法用于获取指定 name 的元素，因为 name 属性可以重复，因此返回一个 HTMLCollection 对象，该对象包含所有匹配的元素。

getElementsByTagName

getElementsByTagName 方法用于获取指定标签名的元素，同理也会返回一个 HTMLCollection 对象，该对象包含所有匹配元素。

DOM 对象常用属性和方法

方法 作用
appendChild(child) 添加子节点
insertBefore(newNode, referenceNode) 在指定节点之前插入新的节点
replaceChild(newChild, oldChild) 替换子节点
属性 作用
childNodes 获取所有子节点
firstChild 获取第一个子节点
lastChild 获取最后一个子节点
parentNode 获取父节点
nextSibling 获取下一个兄弟节点
previousSibling 获取上一个兄弟节点
className 获取或设置元素的 class 属性
innerHTML 获取或设置元素的内容
innerText 获取或设置元素内容

正则表达式对象

正则表达式对象是 JavaScript 中的一个内置对象，用于匹配字符串。正则表达式对象提供了一系列方法，用于匹配字符串，并返回匹配结果。

正则表达式的编写方式有两种：

  1. 字符串模式：使用 new RegExp(pattern, modifiers) 创建正则表达式对象，如：const reg = new RegExp("[0-9]", "g");
  2. 字符串模式：使用 /pattern/modifiers 创建正则表达式对象，如：const reg = /[0-9]/g;

正则表达式对象提供了以下方法：

方法 描述
exec() 匹配字符串，返回匹配结果
test() 匹配字符串，返回布尔值
