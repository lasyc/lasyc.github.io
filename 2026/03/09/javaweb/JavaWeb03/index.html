jQuery 介绍

jQuery 是一个开源的 JavaScript 库，它简化了 JavaScript 的开发，并使 JavaScript 更易用。

jQuery 初体验

jQuery 核心函数

$ 函数是 jQuery 的核心函数，用于创建 jQuery 对象。根据传入的参数不同，$ 函数会返回不同的对象。

参数类型 功能 示例
CSS 选择器字符串 选择 DOM 元素 $('div')
HTML 字符串 创建新 DOM 元素 $('<p>Hello</p>')
DOM 元素/数组 封装为 jQuery 对象 $(document.body)
函数 文档加载就绪回调 $(function(){})
jQuery 对象 复制 jQuery 对象 $(existingjQueryObj)
选择器 + 上下文 在上下文中选择 $('li', 'ul')
无参数/无效参数 空 jQuery 对象 $()
window/document 对象 操作窗口/文档 $(window).scrollTop()

区分 DOM 对象和 jQuery 对象

通过 document.getElementByXXX() 获取的是 DOM 对象，通过 $('xxxx') 获取的 jQuery 对象。

jQuery 对象的本质

jQuery objects 是 DOM 节点的集合，jQuery 集合中的每个元素都是 DOM 节点。

jQuery 对象和 DOM 对象使用区别

jQuery 对象和 DOM 对象在方法和属性上有所不同。

相互转换

我们可以通过 $(DOM对象) 将 DOM 对象转为 jQuery 对象，或者通过 jQuery对象[0] 将 jQuery 对象转为 DOM 对象。

jQuery 基础选择器

选择器 描述 示例
元素选择器 选取所有指定元素 $(“p”)
ID 选择器 选取指定 ID 的元素 $(“#para1”)
类选择器 选取同一类 CSS 的元素 $(“.test”)

层级选择器

选择器 描述 示例
后代选择器 选取所有匹配元素之后的所有元素 $(“div p”)
子选择器 选取父元素下的所有直接子元素 $(“div > p”)
兄弟选择器（相邻） 选取相邻的兄弟元素 $(“div + p”)
兄弟选择器（所有） 选取所有兄弟元素 $(“div ~ p”)

筛选选择器

基本

选择器 描述 示例
:first 选取第一个元素 $(“p:first”)
:last 选取最后一个元素 $(“p:last”)
:even 选取偶数 $(“p:even”)
:odd 选取奇数 $(“p:odd”)
:eq() 选取索引为 n 的元素 $(“p:eq(1)”)
:gt() 选取索引大于 n 的元素 $(“p:gt(1)”)
:lt() 选取索引小于 n 的元素 $(“p:lt(1)”)
:not() 选取所有，除了 n 元素 $(“p:not(1)”)
:header 选取所有标题元素 $(“:header”)
:animated 选取所有正在执行动画的元素 $(“:animated”)

内容

选择器 描述 示例
:contains() 选取包含指定文本的元素 $(“:contains(‘Hello’)”)
:empty 选取所有没有子元素的元素 $(“:empty”)
:has() 选取包含指定子元素的元素 $(“:has(p)”)
:parent 选取所有有子元素的元素 $(“:parent”)

属性

选择器 描述 示例

jQuery 遍历函数

jQuery 遍历函数包括了用于筛选、查找和串联元素的方法。

函数 描述
.add() 将元素添加到匹配元素的集合中。
.andSelf() 把堆栈中之前的元素集添加到当前集合中。
.children() 获得匹配元素集合中每个元素的所有子元素。
.closest() 从元素本身开始，逐级向上级元素匹配，并返回最先匹配的祖先元素。
.contents() 获得匹配元素集合中每个元素的子元素，包括文本和注释节点。
.each() 对 jQuery 对象进行迭代，为每个匹配元素执行函数。
.end() 结束当前链中最近的一次筛选操作，并将匹配元素集合返回到前一次的状态。
.eq() 将匹配元素集合缩减为位于指定索引的新元素。
.filter() 将匹配元素集合缩减为匹配选择器或匹配函数返回值的新元素。
.find() 获得当前匹配元素集合中每个元素的后代，由选择器进行筛选。
.first() 将匹配元素集合缩减为集合中的第一个元素。
.has() 将匹配元素集合缩减为包含特定元素的后代的集合。
.is() 根据选择器检查当前匹配元素集合，如果存在至少一个匹配元素，则返回 true。
.last() 将匹配元素集合缩减为集合中的最后一个元素。
.map() 把当前匹配集合中的每个元素传递给函数，产生包含返回值的新 jQuery 对象。
.next() 获得匹配元素集合中每个元素紧邻的同辈元素。
.nextAll() 获得匹配元素集合中每个元素之后的所有同辈元素，由选择器进行筛选（可选）。
.nextUntil() 获得每个元素之后所有的同辈元素，直到遇到匹配选择器的元素为止。
.not() 从匹配元素集合中删除元素。
.offsetParent() 获得用于定位的第一个父元素。
.parent() 获得当前匹配元素集合中每个元素的父元素，由选择器筛选（可选）。
.parents() 获得当前匹配元素集合中每个元素的祖先元素，由选择器筛选（可选）。
.parentsUntil() 获得当前匹配元素集合中每个元素的祖先元素，直到遇到匹配选择器的元素为止。
.prev() 获得匹配元素集合中每个元素紧邻的前一个同辈元素，由选择器筛选（可选）。
.prevAll() 获得匹配元素集合中每个元素之前的所有同辈元素，由选择器进行筛选（可选）。
.prevUntil() 获得每个元素之前所有的同辈元素，直到遇到匹配选择器的元素为止。
.siblings() 获得匹配元素集合中所有元素的同辈元素，由选择器筛选（可选）。
.slice() 将匹配元素集合缩减为指定范围的子集。

属性的操作

attr 和 prop 虽然均可以获取属性和设置属性，但是 attr 无法操作布尔类型的属性，如 checked、selected、disabled，因为会返回 undefined，而 prop 可以操作布尔类型的属性可以返回 true/false，但是 attr 可以操作自定义属性。

toggleClass() 从匹配的元素中添加或删除一个类。
val() 设置或返回匹配元素的值。

事件的冒泡

事件冒泡是指，当一个元素上的事件触发时，会向它的父元素传播，直到document。我们可以通过event.stopPropagation()来阻止事件冒泡。

jQuery 事件对象

jQuery 事件对象 event 是一个对象，封装了事件触发时浏览器提供的所有信息。如果函数无参数，则 event 可以省略，若函数有参数，则 event 必须放在参数列表的第一个位置。
