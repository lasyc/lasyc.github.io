XML 简介

什么是XML

XML（Extensible Markup Language，可扩展标记语言）是一种用于存储和传输数据的结构化语言。XML 的主要作用是传输数据，比如：HTML、CSS、JavaScript、PHP、ASP、JSP 等。

作用

XML 是一种用于存储和传输数据的结构化语言。

XML 也可做为配置文件使用。

XML 语法

文档声明

XML 文档声明一般放在文档的开头，用于声明文档的版本和编码方式。

文档注释

XML 文档注释，也叫 XML 注释，是 XML 文档的注释，用于描述 XML 文档的内容。

元素(标签)

XML 元素是 XML 文档中的最小单位，用于描述 XML 文档中的内容。同 HTML 元素一样，XML 元素也分为单标签和双标签，也可以拥有属性和子元素。

但是 XML 元素没有固定元素名，开发人员可以自定义元素名。

  1. 元素名称区分大小写
  2. 元素名称必须以字母或下划线开头
  3. 元素名称不能以字母 xml（或 XML、Xml 等）开头
  4. 元素名称可以包含字母、数字、连字符、下划线和句点
  5. 元素名称不能包含空格

属性

XML 元素可以有属性。属性用于为元素提供额外的信息。

尽可能使用元素来描述数据，而不是使用属性。

特殊字符

同 HTML 一样，在 XML 中使用特殊字符时，需要使用实体引用。

如果存在大量的特殊字符，可以使用 CDATA 节来处理。CDATA 节是一个包含特殊字符的文本块，它可以被 XML 解析器忽略。CDATA 节以 <![CDATA[ 开始，以 ]]> 结束。

XML 解析

XML 解析器是 XML 文档的解析器，它可以将 XML 文档解析为树结构，并返回结果。

在 Java 中，可以使用 DOM 和 SAX 解析 XML 文档。但是这两种技术已经过时。

  1. DOM 技术是 W3C 组织推荐的 XML 解析技术。几乎所有的编程语言都提供了 DOM 解析器。
  2. SAX 技术是 Java 官方对 DOM 技术的改进。

我们只需要使用已经编写好的第三方技术 dom4j 就可以完成 XML 解析了。
