EL 表达式

EL 表达式，即：Expression Language，表达式语言。

El 表达式，是 JSP 语言中的表达式，用于替换 JSP 语言中的脚本，来进行数据显示。相比于 JSP 脚本，EL 语言更加简洁，更加方便。

EL 表达式范围

EL 表达式，范围有：pageContext、request、session、application。

在编写 EL 表达式时，会依次范围从小到大进行查找 key 对应的 value。

EL 表达式运算符

关系运算符

运算符 描述 示例 结果
== 或 eq 等于 ${1 == 1} 或者 ${1 eq 1} true
!= 或 ne 不等于 ${1 != 1} 或者 ${1 ne 1} false
< 或 lt 小于 ${1 < 1} 或者 ${1 lt 1} false
> 或 gt 大于 ${1 > 1} 或者 ${1 gt 1} false
<= 或 le 小于等于 ${1 <= 1} 或者 ${1 le 1} true
>= 或 ge 大于等于 ${1 >= 1} 或者 ${1 ge 1} true

逻辑运算符

运算符 描述 示例 结果
&& 或 and 逻辑与 ${1 && 1} 或者 ${1 and 1} true
| | 或 or 逻辑或 ${1 | | 1} 或者 ${1 or 1} true
! 或 not 逻辑非 ${!1} 或者 ${not 1} false

算数运算符

运算符 描述 示例 结果
+ 加法 ${1 + 1} 2
- 减法 ${1 - 1} 0
* 乘法 ${1 * 1} 1
/ 或 div 除法 ${1 / 1} 或 ${1 div 1} 1
% 或 mod 取余数 ${1 % 1} 或 ${1 mod 1} 0
** 幂运算 ${1 ** 1} 1

其他

  • emmpty 运算符：判断对象是否为空，如：null、空串、空数组、空集合、空Map。

  • 三元运算符：${条件 ? 值1 : 值2}

  • .运算符：访问对象属性

  • []运算符：访问数组下标对应元素，访问 map 对应的 key 对应元素，如: ${map[key]}

隐藏对象

在 EL 表达式中，存在着 11 个隐藏对象，可以直接使用

隐藏对象 描述
pageContext 用于获取 JSP 中的九个内置对象
pageScope 用于获取 pageContext 域中数据
requestScope 用于获取 request 域中对象
sessionScope 用于获取 session 域中对象
applicationScope 用于获取 application 域中对象
param 用于获取请求参数
paramValues 用于获取请求参数数组
header 用于获取请求头
headerValues 用于获取请求头数组
cookie 用于获取 Cookie
initParam 用于获取 web.xml 中配置的参数

域对象

pageScope、requestScope、sessionScope、applicationScope 这四个对象，分别对应 JSP 中的 pageContext、request、session、application 对象。通过 . 运算符，可以访问域对象中的属性。

pageContext

pageContext 对象，是 JSP 中最常用的对象，它封装了 JSP 的九个内置对象，并且提供了一些方法，用于获取 JSP 中的对象。

通过 EL 表达式，我们可以直接通过 . 运算符，访问 pageContext 对象中的属性和方法。而不是 getXXX。
