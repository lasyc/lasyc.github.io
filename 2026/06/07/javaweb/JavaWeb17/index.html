什么是 AJAX

AJAX 是 Asynchronous JavaScript and XML 的缩写，中文名称为 异步 JavaScript 和 XML，是一种在不重新加载整个页面的情况下与服务器交换数据并更新部分网页的技术。

示例

  1. 创建 XMLHttpRequest 对象
  2. 设置请求参数
  3. 发送请求
  4. 获取响应数据

JQuery 中的 AJAX

在 JQuery 中，发送 AJAX 请求非常简单，可以使用 $.ajax() 方法，或者更简化的 $.get() 和 $.post() 方法。

在 $.get() 和 $.post() 方法的基础上，还有 $.getJSON() 方法，专门用于获取 JSON 数据，参数顺序与，$.getJSON() 方法只适用于 GET 请求。

序列化 serialize()

在 JQuery 中，如果要发送表单数据，可以使用 serialize() 方法将表单数据序列化成 URL 编码的字符串，也就是 key=value&key=value 的形式。
