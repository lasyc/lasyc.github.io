日语语法总结

动词分类

分类 特点 示例
一类动词 う段结尾 / あ段/う段/お段 + る 書く、読む、話す、買う
二类动词 い段/え段 + る 食べる、見る、起きる
三类动词（サ変） する结尾 勉強する、掃除する
三类动词（カ変） 来る 来る（くる）

注意：特例——切る（きる）、帰る（かえる）、走る（はしる）虽然る前面是い段或え段，但属于一类动词

动词变形总结

ます形

动词类型 变形规则 示例
一类动词 う段 → い段 + ます 書く → 書きます
二类动词 る → ます 食べる → 食べます
サ変 する → します 勉強する → 勉強します
カ変 来る → 来ます（きます） 来る → 来ます
时态 形式
现在/将来 ます
过去 ました
现在/将来否定 ません
过去否定 ませんでした

ます形相关语法

语法 含义 示例
～ます形去掉ます + たい 想做某事 食べたい（想吃）
～ませんか 委婉提议，一起做~好吗 映画を見ませんか
～ましょう 邀请对方做某事 一緒に行きましょう
～ます形去掉ます + ながら 一边一边 音楽を聞きながら勉強します
～ます形去掉ます + 方 ~的做法 読み方（读法）、書き方（写法）
～ます形去掉ます + やすい/にくい 容易/难以 書きやすい（容易写）
～ます形去掉ます + はじめます/出します 开始~ 雨が降り始めました
～ます形去掉ます + 続けます 持续~ 使い続けていました
～ます形去掉ます + 終わります 结束~ 読み終わりました
～ます形去掉ます + に 行きます/来ます 去做 泳ぎに行きます
～ます形去掉ます + やすい/にくい/がたい 做……是容易的/不容易的/不可能的

て形

动词类型 变形规则 示例
一类动词 う、つ、る → って 買う → 買って
一类动词 ぬ、ぶ、む → んで 読む → 読んで
一类动词 く → いて 書く → 書いて
一类动词 ぐ → いで 泳ぐ → 泳いで
一类动词 す → して 話す → 話して
一类动词 行く → 行って（特例） 行く → 行って
二类动词 る → て 食べる → 食べて
サ変 する → して 勉強する → 勉強して
カ変 来る → 来て（きて） 来る → 来て

て形相关语法

语法 含义 示例
～てください 请求对方做某事 窓を開けてください
～てくださいませんか 同上，更礼貌 少し待ってくださいませんか
～ている 正在进行/习惯/状态 今、ご飯を食べています
～が/は + 他动词て形 + ある 有目的地完成某动作，结果保持 ドアが開けてある
～てから 先做A再做B 食べてから、勉強します
～ても 即使也 雨が降っても、行きます
～てはいけません 禁止做某事 ここで遊んではいけません
～てもいいですか 可以做~吗 写真を撮ってもいいですか
～ていきます/きます 移动方向/状态变化 持っていきます
～てしまいます 不愉快的结果/完成 忘れてしまいました
～てみます 尝试做某事 食べてみます
～ておきます 为~做准备 予約しておきます
～てほしい 想让别人做某事 早く来てほしい
～ているところです 正处于~阶段 論文を書いているところです
～たばかりです 刚做完~ 先月中国に来たばかりです
～て始めます/出します 开始~ 雨が降り始めました
～て続けます 持续~ 使い続けていました
～て終わります 结束~ 読み終わりました
～てある 表示动作、作用的结果的持续
～ている 表示动作正在进行，或表示平时反复进行的某型行为习惯，表示状态的持续
～ておく/とく 事先……
～ていく 朝着远离自己的方向而去（空间上），表示某个工作未来会持续下去
～てくる 去去就来，表示出去后还会回来、朝着靠近自己的方向而来、动作一直持续直到现在
～てしまう/ちゃう ……完了，表示动作已经结束，常与「もう」搭配使用
～てみる 试着……看，表示尝试做某事
～ても 不管……都、就算……也
～てもかまわない ……也行、……也可以

た形

变形规则和て形一样，て → た、で → だ

た形相关语法

语法 含义 示例
～たほうがいい 最好做，做比较好 早く寝たほうがいい
～たあとで 在~之后 洗濯したあとで、映画を見ます
～たことがあります 做过某事，有过经历 日本へ行ったことがあります
～たところです 刚做完~ 家に帰ったところです
～たら 假定条件，如果~ 雨が降ったら、行きません
～たまま 保持着~状态做另一件事 テレビをつけたまま、出かけました
～たり～たりします 有时有时 日曜日は、買い物したり、映画を見たりします

ない形

动词类型 变形规则 示例
一类动词 う段 → あ段 + ない 書く → 書かない
一类动词 う → わ（特例） 買う → 買わない
二类动词 る → ない 食べる → 食べない
サ変 する → しない 勉強する → 勉強しない
カ変 来る → 来ない（こない） 来る → 来ない

注意：「ある」的ない形是「ない」（特殊）

ない形相关语法

语法 含义 示例
～ないでください 请求对方不要做某事 ここに車を停めないでください
～ないでくださいませんか 同上，更礼貌 もう言わないでくださいませんか
～ないで 不做A就做了B 朝ごはんを食べないで、出かけました
～なければなりません 必须做某事 勉強しなければなりません
～ないといけません 同上 勉強しないといけません
～なくてもいいです 不做~也可以 勉強しなくてもいいです
～なければ 必须~ 早く行かなければ
～ないで/ずに 不就 傘を持たずに出かけました
～なくてはならない/なくてはいけない 必须……

注意：「ないで」和「なくて」的区别——有因果关系用「なくて」，否则用「ないで」

命令形

动词类型 变形规则 示例
一类动词 う段 → え段 読む → 読め
二类动词 る → ろ 食べる → 食べろ
サ変 する → しろ 勉強する → 勉強しろ
カ変 来る → こい 来る → 来い

命令形相关语法

语法 含义 示例
～なさい 命令（比命令形客气） 勉強しなさい
～な 禁止做某事 触るな（不要碰）
～て/ないで 祈使说法 早く来て

意志形

动词类型 变形规则 示例
一类动词 末尾音节 → お段 + う 書く → 書こう
二类动词 る → よう 食べる → 食べよう
サ変 する → しよう 勉強する → 勉強しよう
カ変 来る → 来よう（こよう） 来る → 来よう

意志形相关语法

语法 含义 示例
～と思います 我觉得/想~（第一人称） 行こうと思います
～と思っています 持续的想法（第二/三人称） 行こうと思っています
～ましょう 提议做~ 一緒に行きましょう

ば形

动词类型 变形规则 示例
一类动词 末尾音节 → え段 + ば 書く → 書けば
二类动词 る → れば 食べる → 食べれば
サ変 する → すれば 勉強する → 勉強すれば
カ変 来る → 来れば（くれば） 来る → 来れば

ば形相关语法

语法 含义 示例
～ば～ほど 越越 この本を読めば読むほど、面白いです

可能形

动词类型 变形规则 示例
一类动词 末尾音节 → え段 + る 書く → 書ける
二类动词 る → られる 食べる → 食べられる
サ変 する → できる 勉強する → 勉強できる
カ変 来る → 来られる 来る → 来られる

注意：可能形的宾语助词用「が」而不是「を」

可能形相关语法

语法 含义 示例
～ことができます 能做某事 日本語を話すことができます

被动形

动词类型 变形规则 示例
一类动词 末尾音节 → あ段 + れる 読む → 読まれる
二类动词 る → られる 食べる → 食べられる
サ変 する → される 勉強する → 勉強される
カ変 来る → 来られる（こられる） 来る → 来られる

被动形相关语法

语法 含义 示例
～は～に～れます 被~ 馬さんは森さんにカメラを壊されました
～は～によって～れます 由~（强调施动者） この車は日本のデザイナーによって設計されました

使役形

动词类型 变形规则 示例
一类动词 末尾音节 → あ段 + せる 読む → 読ませる
二类动词 る → させる 食べる → 食べさせる
サ変 する → させる 勉強する → 勉強させる
カ変 来る → 来させる（こさせる） 来る → 来させる

使役形相关语法

语法 含义 示例
～は～を～せます 让做 先生は学生を走らせます
～は～に～を～せます 让做（他动词） 先生は学生に本を読ませます
～せてください 请求允许自己做~ 行かせてください

使役被动（使役 → 被动）

动词类型 变形规则 示例
一类动词 末尾音节 → あ段 + せられる（される） 飲まされる（被迫喝）
二类动词 る → させられる 食べさせられる（被迫吃）
サ変 する → させられる 勉強させられる（被迫学习）
カ変 来る → 来させられる 来させられる

形容词变形总结

一类形容词（い形容词）

形式 变形规则 示例
肯定 い形容词 + です 高いです
否定 い → くない/くありません 高くない
过去式 い → かった 高かった
过去否定 い → なかった 高くなかった
て形 い → くて 高くて
た形 い → かった 高かった
ば形 い → ければ 高ければ
修饰名词 い形容词 + 名词 高い山
修饰动词 い → く + 动词 早く帰ります

注意：「いい」的原形是「よい」，否定是「よくない」，过去式是「よかった」

二类形容词（な形容词）

形式 变形规则 示例
肯定 二类形容词 + です 静かです
否定 二类形容词 + ではありません 静かではありません
过去式 二类形容词 + でした 静かでした
过去否定 二类形容词 + ではありませんでした 静かではありませんでした
て形 二类形容词 + で 静かで
た形 二类形容词 + だった 静かだった
ば形 二类形容词 + であれば 静かであれば
修饰名词 二类形容词 + な + 名词 静かな部屋
修饰动词 二类形容词 + に + 动词 部屋をきれいに掃除する

注意：綺麗（きれい）、詳細（しょうさい）、愉快（ゆかい）、嫌い（きらい）虽然以い结尾，但是二类形容词

名词

形式 示例
肯定 学生です
否定 学生ではありません
过去式 学生でした
过去否定 学生ではありませんでした
修饰名词 名词 + の + 名词

注意：名词和二类形容词变形规则一致

形容词变化相关语法

语法 含义 示例
一类形 + く + なります 变得~（自然变化） 暑くなりました
一类形 + く + します 使~变得（人为变化） 音を大きくします
二类形/名 + になります 变得~（自然变化） 静かになりました
二类形/名 + にします 使~变得（人为变化） 部屋をきれいにします
～たほうがいい 最好做~ 早く寝たほうがいい
～ないほうがいい 最好不做~ 無理をしないほうがいい
～そうです 看起来~（推测） 雨が降りそうです
～そうです 听说~（传闻） 雨が降るそうです
～ます形去掉ます + 過ぎます 过于~ 食べ過ぎました

副词总结

程度副词

副词 含义 示例
とても/たいへん 非常 この映画はとても面白いです（这部电影非常有趣）
かなり 出乎意料 この問題はかなり難しいです（这个问题出乎意料地难）
少し/ちょっと 一点、稍微 少し寒いです（有点冷）/ ちょっと待ってください（稍等一下）
なかなか 相当/怎么也 肯定：この本はなかなか面白いです（这本书相当有趣）；否定：この問題はなかなか分かりません（这个问题怎么也不明白）
だんだん 逐渐 日本語がだんだん分かるようになりました（日语渐渐能听懂了）
あまり 不怎么~ 日本料理はあまり食べません（不太吃日本菜）
全然 完全不~ 日本語は全然分かりません（完全不懂日语）

频率副词

副词 含义 示例
いつも 总是 いつも朝ごはんを食べます（总是吃早饭）
よく 经常 よくここで勉強します（经常在这里学习）
時々 有时 時々映画を見ます（有时看电影）
たまに 偶尔 たまに友達と遊びます（偶尔和朋友玩）
あまり ~ません 不怎么~ あまり運動しません（不怎么运动）
全然 ~ません 完全不~ 全然食べませんでした（完全没吃）

其他常用副词

副词 含义 示例
もう 已经/再 もう食べました（已经吃了）/ もう一つください（请再给一个）
まだ 还/还没 肯定：まだ食べています（还在吃）；否定：まだ食べていません（还没吃）
ぜひ 一定、务必 ぜひ日本へ行ってください（请一定去日本）
絶対（に） 一定、肯定 絶対に来ます（一定来）
せっかく 特意、好不容易 せっかく来ていただいたのに（好不容易来一趟……）
つい 无意中 つい食べてしまいました（不小心吃多了）

常用语法汇总

表达原因

语法 特点 示例
～から 主观原因（个人决定、判断） 暑いですから、窓を開けます
～ので 客观原因（事实、自然现象） 雨なので、傘を持っていきます
～て 原因（口语常用） 頭が痛くて、今日は休みます
～ために 原因（不情愿的事、坏事） 病気のために、休みました

表达目的

语法 含义 示例
～ために 为了~（意志性动作） 日本語を勉強するために、日本へ行きました（为了学日语，去了日本）
～ように 为了~（非意志性/努力做到） 遅刻しないように、早く起きます（为了不迟到，早起）

表达推测

语法 特点 示例
～でしょう/だろう 主观推测 明日は雨でしょう（明天可能下雨吧）
～かもしれません 有可能（可能性较低） 明日は雨が降るかもしれません（明天也许会下雨）
～はずです 根据理由推断（有把握） もう届いたはずです（应该已经到了）
～はずがありません 不可能~ そんなことあるはずがありません（不可能有那种事）
～そうです（推测） 看起来~ 雨が降りそうです（看起来要下雨）
～そうです（传闻） 听说~ 雨が降るそうです（听说要下雨）
～ようです 根据感知推测（委婉） 雨が降るようです（好像要下雨）
～みたいです 同上（较口语） 雨が降るみたいです（好像要下雨）
～らしい 根据观察推测（不强调来源） 雨が降るらしいです（听说/好像要下雨）

表达义务/许可

语法 含义 示例
～なければなりません 必须~ 勉強しなければなりません
～ないといけません 同上 勉強しないといけません
～なくてもいいです 不~也可以 勉強しなくてもいいです
～てもいいですか 可以~吗 写真を撮ってもいいですか

表达变化

语法 含义 示例
～になります 变得~（自然） 元気になりました
～にします 使~变得（人为） 部屋をきれいにします
～くなります 变得~（い形容词，自然） 暖かくなりました
～くします 使~变得（い形容词，人为） 音を大きくします

表达愿望/决定

语法 含义 示例
～たいです 想做~ 日本へ行きたいです
～ほしい 想要~（物品） 新しいパソコンがほしい
～つもりです 打算~ 今年中にお金を貯めるつもりです
～ことにします 自己决定~ 毎日運動することにしました
～ことになりました 被决定~（外在原因） 来月から新しいプロジェクトに参加することになりました

表达评价/比较

语法 含义 示例
～は～より～です 比更~ 夏は冬より暑いです（夏天比冬天热）
～のほうが～ 更 冬のほうが好きです（更喜欢冬天）
～ほど～ない 没有那么~ 夏ほど暑くないです（没有夏天那么热）
～がいちばん～ 是最的 季節の中で秋がいちばん好きです（季节中最喜欢秋天）

表达列举/限定

语法 含义 示例
～と～ 完全列举（≤3个） りんごとバナナを買いました
～や～など 部分列举 りんごやバナナなどを買いました
～か 选择/不确定 コーヒーか紅茶、どちらがいいですか
～だけ 仅、只（中性） 三人だけ来ました
～しか～ない 仅、只（消极） 三人しか来ませんでした
～でも 即使也/极端例子 子供でもできます
～ばかり 净是、全是 野菜ばかり食べています

表达时间

语法 含义 示例
～前に 在~之前 食べる前に、手を洗ってください
～あとで 在~之后 洗濯したあとで、映画を見ます
～たところです 刚做完~ 家に帰ったところです
～たばかりです 刚做完~（时间更长） 先月中国に来たばかりです
～とき 在~时 学生のとき、よく遊んでいた
～ています 正在/习惯/状态 毎日走っています

表达语气

语法 含义 示例
～よ 提醒、告知 これは私のですよ
～ね 感叹、征求同意 今日はいい天気ですね
～よね 提出意见征求同意 明日は休みですよね
～かな 不确定 明日は雨かな
～って 传闻 明日は休みだって
～けど 转折 用事があるんですけど

表达条件

语法 特点 示例
～と 必然条件 ボタンを押すと、電源が入ります
～たら 假定条件 雨が降ったら、行きません
～ば 假定条件 安ければ、買います
～たら 与「ば」区别 「たら」更口语，条件更具体

表达被动/使役

语法 含义 示例
～は～に～れます 被~ 馬さんは森さんにカメラを壊されました
～は～を～せます 让做 先生は学生を走らせます
～は～に～を～せます 让做（他动词） 先生は学生に本を読ませます
～せてください 请求允许做~ 行かせてください
～させられます 被迫~ 勉強させられます

表达自他动词

语法 含义 示例
名 + を + 他动词 强调动作 電気を消します（把灯关了）
名 + が + 自动词 描述状态 電気が消えました（灯关着）
自动词 + ている 状态持续 窓が開いている（窗户开着）
他动词 + てある 有意动作的结果保持 窓が開けてある（窗户开着/有人开着）

易混淆语法对比

一、表达原因：～から vs ～ので vs ～て vs ～ために

语法 语气/来源 使用场景 能否用于句尾 示例
～から 主观（个人决定、判断） 强调个人意志 ✅ 可以 暑いから、窓を開けます
～ので 客观（事实、自然现象） 陈述客观事实 ❌ 只能句中 雨なので、傘を持っていきます
～て 口语常用 日常会话 ❌ 只能句中 頭が痛くて、今日は休みます
～ために 不情愿/坏事 常用于消极原因 ❌ 只能句中 病気のために、休みました

区分要点：

  • 对长辈/正式场合 → ～ので（礼貌）
  • 对朋友/家人 → ～から / ～て（随意）
  • 坏事/不情愿 → ～ために（如生病、受伤、给他人添麻烦等）
  • 句尾表达原因 → 只能用 ～から（如：暑いですから、窓を開けてください）

二、～ために vs ～ように（目的）

语法 前接形式 前项特点 后项特点 示例
～ために 动词基本形 / 名词 + の 意志性动作 可以是意志性动作 日本語を勉強するために、日本へ行きました
～ように 动词基本形 / ない形 非意志性动作/状态 常搭配 ～ようにします（努力做到） 忘れないように、メモします

区分要点：

  • 前项有明确意志 → ～ために
  • 前项是非意志性（如”不忘”、”不迟到”）→ ～ように
  • 后项是”努力做到某事” → ～ようにします（如：早く起きられるようにします）

三、～ても vs ～てもいいですか（让步 vs 许可）

语法 含义 示例
～ても 即使也（让步） 雨が降っても、行きます（即使下雨也去）
～てもいいですか 可以~吗（请求许可） 写真を撮ってもいいですか（可以拍照吗？）

注意区分 ～ても 和 ～てもいいですか：

语法 含义 示例
～ても 即使也（让步） 高くっても、買います（即使贵也买）
～てもいいですか 可以~吗（请求许可） 写真を撮ってもいいですか（可以拍照吗？）

四、～ている 的三种含义

用法 含义 动词特点 示例
进行中 正在做~ 瞬间动词/持续动词 今、ご飯を食べています（正在吃饭）
状态持续 ~着 瞬间动词（变化后状态） ドアが開いています（门开着）
习惯/反复 经常做~ 持续动词 每日走っています（每天跑步）

区分要点：

  • 瞬间动词 + ている → 通常是状态持续（如：结过婚 → 結婚している）
  • 持续动词 + ている → 通常是进行中或习惯
  • 判断方法：看是否能在句中加”今”（现在），能加则是进行中

五、～たところです vs ～たばかりです（刚做完）

语法 时间长短 使用限制 示例
～たところです 很短（几秒~几小时前） 不能用”先月”等长时间 中国に来たところです（刚到中国）/ ご飯を食べたところです（刚吃完饭）
～たばかりです 较长（几小时~几个月前） 可以用”先月”等长时间 先月中国に来たばかりです（上月刚到中国）/ 卒業したばかりです（刚毕业）

区分要点：

  • 下飞机/刚吃完饭 → ～たところです
  • 上个月刚毕业/去年刚结婚 → ～たばかりです

六、～そうです 两种含义

用法 接续方式 含义 示例
推测（样态） 动词ます形去掉ます + そうです / い形容词去掉い + そうです / な形容词直接加 看起来~ 雨が降りそうです（看起来要下雨）
传闻 动词简体形 + そうです / な形容词词干 + そうです / 名词 + そうです 听说~ 雨が降るそうです（听说要下雨）

区分要点：

  • 动词ます形 + そうです → 推测（样态）→ “看起来要~”
  • 动词简体形 + そうです → 传闻 → “听说~”
  • な形容词：静かそうです（看起来安静）vs 静かそうです（听说很安静）→ 需要根据语境判断

七、～ようです vs ～らしいです vs ～みたいです（推测）

语法 信息来源 语气 使用场景 示例
～ようです 自己的感知（看/听/感觉） 委婉、客气 正式场合/对被观察者 風邪を引いたようですね（你好像感冒了）
～みたいです 同上 较口语 日常会话 風邪を引いたみたいですね
～らしいです 有信息来源但不强调 较客观 转述他人信息 風邪を引いたらしいです（好像感冒了/听说感冒了）

区分要点：

  • 和被观察者本人说话 → 只能用 ～ようです（如：あなたは風邪を引いたようですね）
  • 转述别人说的话 → ～らしいです
  • ～ようです 和 ～みたいです 基本相同，只是 ～みたいです 更口语

八、～に vs ～で（场所）

语法 动词特点 含义 示例
～に 静态动词（ある、いる、住む等） 存在的位置 机の上に本がある（桌子上有书）/ 家にいます（在家）
～で 动态动词（勉強する、食べる等） 动作进行的场所 図書館で勉強します（在图书馆学习）/ レストランで食べます（在餐厅吃饭）

区分要点：

  • 表示”在哪里”（存在）→ ～に
  • 表示”在哪里做某事”（动作）→ ～で
  • 注意：に还可以表示时间、目的地、对象等

九、～に vs ～へ（目的地）

语法 侧重点 示例
～に 强调目的（到达点） 学校に行きます（去学校）
～へ 强调方向（朝某个方向移动） 学校へ行きます（往学校去）

区分要点：

  • 两者几乎可以互换，但 ～に 更强调”到达”，～へ 更强调”方向”
  • ～に 还有其他用法（时间、对象、存在场所等），～へ 只表示方向

十、～は vs ～が（主语）

语法 侧重点 使用场景 示例
～は 主题（已知信息） 对主题进行说明 私は学生です（我是学生）/ 日本語は難しいです（日语很难）
～が 主语（新信息） 强调主语/首次出现 田中さんが来ました（田中来了）/ あそこに猫がいる（那边有只猫）

区分要点：

  • 疑问词做主语 → 用 が（如：誰が来ましたか → 田中さんが）
  • 描述眼前看到的/新信息 → が
  • 谈论已知话题 → は
  • ～は～が～ 结构：豚肉は値段が安い（猪肉价格便宜）

十一、～を vs ～で（手段/工具）

语法 含义 示例
～で 表示手段、方式、工具 バスで学校へ行きます（坐公交去学校）/ パソコンで仕事をします（用电脑工作）
～を 表示动作经过的场所/离开的场所 公園を散歩します（在公园散步）/ 家を出ました（出了家门）

区分要点：

  • 使用什么交通工具/工具 → ～で
  • 经过/离开某地 → ～を（如：公園を散歩する、家を出る）

十二、～まで vs ～までに（截止时间）

语法 含义 后接动词 示例
～まで 到~为止（持续做某事） 持续性动词 五時まで働きます（工作到五点）
～までに 在~之前（完成某事） 完成性动词 五時までに帰ります（五点前回来）

区分要点：

  • 一直做某事到某个时间点 → ～まで（如：看电视看到10点）
  • 在某个时间点之前完成某事 → ～までに（如：10点前交作业）

十三、～だけ vs ～しか～ない vs ～ばかり（限定）

语法 语气 后接肯定/否定 示例
～だけ 中性（客观陈述） 肯定 三人だけ来ました（只来了三个人）
～しか～ない 消极（不满、不够） 否定 三人しか来ませんでした（才来了三个人）
～ばかり 中性/强调多 肯定 野菜ばかり食べています（净吃蔬菜）

区分要点：

  • 客观描述”只有这些” → ～だけ
  • 表达”太少了、不够” → ～しか～ない
  • 表达”全是某类事物” → ～ばかり

十四、～ないで vs ～なくて

语法 含义 使用场景 示例
～ないで 不做A就做了B 无因果关系（附带状况） 朝ごはんを食べないで、出かけました（没吃早饭就出门了）
～なくて 因为没做A，所以B 有因果关系 食べなくて、元気がなかった（因为没吃，所以没精神）

区分要点：

  • 有因果关系（因为没做所以）→ ～なくて
  • 无因果关系（不做就）→ ～ないで
  • 形容词/名词只有 ～なくて，没有 ～ないで（如：この本は面白くなくて、読みませんでした（因为这本书没意思，所以没读））

十五、～てしまう vs ～たことがある

语法 含义 示例
～てしまう 不愉快的结果 / 完成 忘れてしまいました（竟然忘了）/ 読んてしまいました（读完了）
～たことがある 有过某种经历 日本へ行ったことがあります（去过日本）/ 寿司を食べたことがあります（吃过寿司）

区分要点：

  • 表达后悔、遗憾或完成 → ～てしまう
  • 表达经历、体验 → ～たことがある

十六、～ている（进行）vs ～たところ（刚完成）

语法 含义 示例
～ている 正在进行 今、ご飯を食べています（正在吃饭）
～たところ 刚完成 ご飯を食べたところです（刚吃完饭）

区分要点：

  • 正在吃 → ～ている
  • 刚吃完 → ～たところ

十七、～てはいけません vs ～ないでください

语法 语气 示例
～てはいけません 较生硬、严厉 ここで遊んではいけません（不许在这里玩）/ そんなことを言ってはいけません（不许说那种话）
～ないでください 较委婉、礼貌 ここに車を停めないでください（请不要在这里停车）/ 大声で話さないでください（请不要大声说话）

区分要点：

  • 对孩子/严厉禁止 → ～てはいけません
  • 对他人/礼貌请求 → ～ないでください

十八、～ことにする vs ～ことになる

语法 主语 含义 示例
～ことにする 自己 自己决定做~ 毎日運動することにしました（决定每天运动）
～ことになる 外部 被决定/安排做~ 来月から新しいプロジェクトに参加することになりました（下个月开始参加新项目）

区分要点：

  • 自己主动决定 → ～ことにする
  • 被安排/被要求/客观决定 → ～ことになる

十九、～について vs ～にとって

语法 含义 示例
～について 关于~（话题） 日本文化について勉強します（学习关于日本文化的事）
～にとって 对~来说（立场/评价） 私にとって、大切な人です（对我来说，是重要的人）

区分要点：

  • 谈论某个话题/主题 → ～について
  • 从某人的立场/角度来评价 → ～にとって

二十、～けど vs ～が（转折）

语法 语气 示例
～が 较正式 日本語は難しいが、面白いです（日语很难，但很有趣）
～けど 较口语 日本語は難しいけど、面白いです（日语很难，但很有趣）

注意： ～けど 还有铺垫、委婉引入话题的用法（如：すみませんけど、駅はどこですか？（不好意思，请问车站在哪？））

二十一、～と思います vs ～と思っています

语法 人称 时间 示例
～と思います 第一人称 当前想法 明日は休みだと思います（我觉得明天休息）
～と思っています 第二/三人称 持续已久的想法 田中さんは日本へ行きたいと思っています（田中一直想去日本）

区分要点：

  • 说话人自己的想法 → ～と思います
  • 描述他人持续的想法/犹豫不决的状态 → ～と思っています

二十二、～ことが出来る vs ～られる

语法 含义 示例
～ことができる 有能力/有条件做~ 日本語を話すことができます（能说日语）
～られる（可能形） 能够~（更口语） 日本語が話せます（能说日语）

区分要点：

  • ～ことができる 更正式、书面
  • 可能形 更口语、简洁

二十三、～そう（样态）vs ～そう（传闻）vs ～よう（样态）

语法 接续 信息来源 示例
～そうです（样态） 动词ます形 / い形容词去い / な形容词 自己的观察 雨が降りそうです
～そうです（传闻） 动词简体 / な形容词词干 / 名词 他人的话/报道 雨が降るそうです
～ようです 动词简体 / な形容词+な / 名词+の 自己的感知（委婉） 雨が降るようです

区分要点：

  • 看到乌云密布 → 降りそうです（样态：看起来要下雨）
  • 听天气预报说 → 降るそうです（传闻：听说要下雨）
  • 感觉天阴阴的 → 降るようです（委婉推测：好像要下雨）

二十四、～にすみます vs ～にかぎります

语法 含义 示例
～にすみます 以~为满足/到此为止 これですみます（这样就可以了）/ これですみません（不好意思/对不起）
～にかぎります 最好~ / 限定 新鮮なものにかぎります（新鲜的东西最好）/ これにかぎります（就是这个了）

区分要点：

  • ～にすみます：表示”这样就足够了”或道歉
  • ～にかぎります：表示”是最好的”或限定范围（就是了）

二十五、～はず vs ～わけ

语法 含义 示例
～はず 根据理由推断（理应如此） もう届いたはずです（应该已经到了）/ 彼は来るはずです（他应该会来）
～わけ 理所当然/解释说明 そんなわけがない（不可能有那种事）/ だから遅刻したわけです（所以才迟到了）

区分要点：

  • ～はず：说话人认为”理应如此”，但不完全确定
  • ～わけ：强调”理所当然”，或用于解释原因（所以~）
