Validation

Validation 是 Spring 中提供的一种可以通过注解来进行参数校验的接口

实现 Validator 接口

  • 引入依赖
  • 创建测试类
  • 自定义校验类，实现 Validator 接口
  • 测试

基于 @Validation 注解

基础校验

注解 用途 示例
@NotNull 不能为 null @NotNull String name
@NotEmpty 不能为 null 且长度/大小 > 0 @NotEmpty List<String> tags
@NotBlank 不能为 null、不能全是空白 @NotBlank String username
@Null 必须为 null @Null String deletedAt

范围校验

注解 用途 示例
@Min 最小值 @Min(18) int age
@Max 最大值 @Max(100) int age
@Positive 必须为正数 @Positive double price
@PositiveOrZero 必须 ≥ 0 @PositiveOrZero int stock
@Negative 必须为负数 @Negative double temperature
@NegativeOrZero 必须 ≤ 0 @NegativeOrZero int debt
@Range 指定范围（Hibernate 扩展） @Range(min=1, max=120) int age

字符串校验

注解 用途 示例
@Size 长度限制 @Size(min=2, max=20) String name
@Length 长度限制（Hibernate 扩展） @Length(min=6, max=32) String password
@Pattern 正则匹配 @Pattern(regexp=”^1[3-9]\\d{9}$”) String phone
@Email 邮箱格式 @Email String email

数字校验

注解 用途 示例
@Digits 整数和小数位数限制 @Digits(integer=5, fraction=2) BigDecimal price

时间校验

注解 用途 示例
@Past 必须是过去的日期 @Past LocalDate birthday
@Future 必须是未来的日期 @Future LocalDate meetingDate
@PastOrPresent 过去或现在 @PastOrPresent LocalDate createdAt
@FutureOrPresent 未来或现在 @FutureOrPresent LocalDate appointmentDate

其他

注解 用途 示例
@AssertTrue 必须为 true @AssertTrue boolean agreed
@AssertFalse 必须为 false @AssertFalse boolean deleted
